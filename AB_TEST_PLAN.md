# A/B 테스트 계획서

## 목표
**무료 시작하는 고객 수를 늘리기**

---

## 1단계: 문제 정의

### 현재 상황 분석
- **현재 상태**: 메인 페이지에 "지금 시작하기" CTA 버튼이 있음
- **측정 가능한 지표**: 
  - CTA 버튼 클릭률 (CTR)
  - 무료 시작 전환률 (클릭 → 실제 가입 완료)
  - 페이지 이탈률
  - 세션당 체류 시간

### 문제점 파악
1. **CTA 버튼의 효과성 미확인**
   - 현재 버튼이 충분히 눈에 띄는가?
   - 버튼 텍스트가 사용자의 행동을 유도하는가?
   - 버튼 위치가 최적인가?

2. **사용자 동기 부여 부족 가능성**
   - "무료"라는 가치 제안이 명확한가?
   - 사용자가 얻을 수 있는 이점이 명확히 전달되는가?
   - 신뢰를 주는 요소가 있는가?

3. **전환 장벽**
   - 가입 프로세스가 복잡한가?
   - 사용자가 망설이는 이유는 무엇인가?

### 핵심 문제 정의
**"사용자가 CTA 버튼을 클릭하고 실제로 무료로 시작하는 비율이 낮다"**

---

## 2단계: 아이디어 도출

### 아이디어 1: CTA 버튼 텍스트 변경
**변경 전**: "지금 시작하기"
**변경 후 옵션들**:
- "무료로 시작하기"
- "지금 무료로 시작하기"
- "무료 체험 시작하기"
- "30일 무료로 시작하기"

**이유**: "무료"라는 키워드를 명시하여 가치 제안을 명확히 함

---

### 아이디어 2: CTA 버튼 디자인 개선
**변경 전**: 파란색 배경, 기본 스타일
**변경 후 옵션들**:
- 더 큰 버튼 크기
- 더 눈에 띄는 색상 (예: 초록색, 주황색)
- 그라데이션 배경
- 아이콘 추가 (예: 화살표, 체크마크)

**이유**: 시각적 강조로 클릭 유도

---

### 아이디어 3: CTA 버튼 위치 변경
**변경 전**: 페이지 중앙 하단
**변경 후 옵션들**:
- 페이지 상단 (헤더 영역)
- 페이지 중앙 상단 (제목 바로 아래)
- 스티키(고정) 버튼 (스크롤해도 항상 보임)
- 여러 위치에 배치 (상단 + 하단)

**이유**: 사용자가 언제든지 쉽게 접근할 수 있도록 함

---

### 아이디어 4: 추가 인센티브 메시지
**변경 전**: CTA 버튼만 존재
**변경 후 옵션들**:
- 버튼 위/아래에 보조 텍스트 추가
  - "신용카드 불필요"
  - "취소는 언제든지 가능"
  - "첫 30일 무료"
- 신뢰 지표 표시
  - "이미 10,000명이 시작했습니다"
  - "평점 4.8/5.0"

**이유**: 사용자의 우려를 해소하고 신뢰를 높임

---

### 아이디어 5: 페이지 레이아웃 개선
**변경 전**: 단순한 환영 메시지
**변경 후 옵션들**:
- 이점(Benefits) 섹션 추가
- 사용 사례(Case Studies) 추가
- FAQ 섹션 추가
- 소셜 증명(Social Proof) 추가

**이유**: 사용자에게 더 많은 정보를 제공하여 전환 의사 결정 지원

---

## 3단계: 가설 설정

### 가설 1: CTA 버튼 텍스트에 "무료" 키워드 추가
**가설**: 
"CTA 버튼 텍스트를 '지금 시작하기'에서 '무료로 시작하기'로 변경하면, 사용자가 무료라는 가치를 명확히 인지하여 클릭률과 전환률이 증가할 것이다."

**측정 지표**:
- CTA 버튼 클릭률 (CTR)
- 무료 가입 완료율
- 버튼 클릭 후 이탈률

**예상 결과**: 
- 클릭률 20% 증가
- 전환률 15% 증가

---

### 가설 2: CTA 버튼 디자인 강화
**가설**:
"CTA 버튼을 더 크고 눈에 띄는 색상(초록색)으로 변경하고, '무료' 아이콘을 추가하면 시각적 강조로 인해 클릭률이 증가할 것이다."

**측정 지표**:
- CTA 버튼 클릭률 (CTR)
- 버튼 가시성 (스크롤 위치별 클릭 분포)
- 무료 가입 완료율

**예상 결과**:
- 클릭률 25% 증가
- 전환률 10% 증가

---

### 가설 3: CTA 버튼 위치 최적화
**가설**:
"CTA 버튼을 페이지 상단(제목 바로 아래)으로 이동하고, 하단에도 스티키 버튼을 추가하면 사용자가 더 쉽게 접근할 수 있어 클릭률이 증가할 것이다."

**측정 지표**:
- 상단 버튼 클릭률 vs 하단 버튼 클릭률
- 전체 CTA 클릭률
- 무료 가입 완료율

**예상 결과**:
- 클릭률 30% 증가
- 전환률 12% 증가

---

### 가설 4: 신뢰 요소 추가
**가설**:
"CTA 버튼 주변에 '신용카드 불필요', '취소는 언제든지 가능' 같은 보조 메시지를 추가하면 사용자의 우려를 해소하여 전환률이 증가할 것이다."

**측정 지표**:
- CTA 버튼 클릭률 (CTR)
- 클릭 후 가입 완료율 (전환률)
- 페이지 이탈률

**예상 결과**:
- 클릭률 10% 증가
- 전환률 25% 증가 (우려 해소로 인한 높은 전환)

---

### 가설 5: 복합 개선 (텍스트 + 디자인 + 위치)
**가설**:
"CTA 버튼 텍스트를 '무료로 시작하기'로 변경하고, 초록색으로 디자인을 강화하며, 상단과 하단에 모두 배치하면 각 요소의 시너지 효과로 클릭률과 전환률이 크게 증가할 것이다."

**측정 지표**:
- CTA 버튼 클릭률 (CTR)
- 무료 가입 완료율
- 페이지 이탈률
- 세션당 체류 시간

**예상 결과**:
- 클릭률 40% 증가
- 전환률 30% 증가

---

## 우선순위 추천

### 1순위: 가설 1 (텍스트 변경)
- **이유**: 구현이 가장 간단하고, "무료" 키워드의 영향이 클 것으로 예상
- **예상 효과**: 중간 수준의 개선

### 2순위: 가설 4 (신뢰 요소 추가)
- **이유**: 전환률 개선에 직접적인 영향
- **예상 효과**: 전환률 대폭 개선

### 3순위: 가설 2 (디자인 강화)
- **이유**: 시각적 개선으로 클릭률 증가 기대
- **예상 효과**: 클릭률 개선

### 4순위: 가설 3 (위치 최적화)
- **이유**: 사용자 경험 개선
- **예상 효과**: 접근성 향상

### 5순위: 가설 5 (복합 개선)
- **이유**: 여러 요소를 동시에 테스트하면 개별 효과를 파악하기 어려움
- **권장**: 개별 가설 검증 후 진행
